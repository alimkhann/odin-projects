# Library

Demo: [Live site](https://alimkhann.github.io/odin-projects/library)
Screenshot:
![Library app shelves and dialogs](docs/images/library-20260206.png)

Tech: HTML, CSS, JavaScript (ES modules).
Usage: Open `index.html` in a browser.
