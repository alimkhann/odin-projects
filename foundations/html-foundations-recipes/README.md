# HTML Foundations Recipes
Based on The Odin Project's [Foundations Recipes](https://www.theodinproject.com/lessons/foundations-recipes).

This project is a collection of traditional Kazakh recipes, implemented using HTML. Each recipe is presented in a structured format, including ingredients and preparation steps.

## Project Structure

- `index.html`: The main page that lists all the recipes.
- `recipes/`: A directory containing individual HTML files for each recipe.
  - `baursak.html`: A recipe for Baursak, a traditional Kazakh bread.
  - `kazy.html`: A recipe for Kazy, a traditional horse meat sausage.
  - `beshbarmak.html`: A recipe for Beshbarmak, a traditional Kazakh dish made with boiled meat and noodles.
  - `manti.html`: A recipe for Manti, traditional steamed dumplings.
  - `kuyrdak.html`: A recipe for Kuyrdak, a traditional dish made from fried meat and organs.

## How to Use

1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to view the list of recipes.
3. Click on any recipe link to view the detailed recipe.
