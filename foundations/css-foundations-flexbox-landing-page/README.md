# Landing Page with Flexbox

A foundational web project from The Odin Project focused on building a responsive landing page using HTML and CSS Flexbox.

## Project Goal
Create a multi-section landing page that demonstrates core flexbox layout principles for structuring and aligning page elements.

## Key Features
- **Header/Navigation**
- **Hero Section**
- **Content Sections**
- **Footer**

## Technologies
- HTML5
- CSS3 (Flexbox)

## Learning Outcomes
- Understanding flexbox
- Alignment and justification of elements

## How to View
Open `index.html` in a web browser.
